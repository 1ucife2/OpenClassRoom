Git exercices

Commande git initialisation

  • git config --global user.name "John Doe"
  • git config --global user.email johndoe@example.com
  • git config --global color.diff auto
  • git config --global color.status auto
  • git config --global color.branch auto
  • git config --global core.editor notepad++
  • --global merge.tool vimdiff
  • git init

Relier un dépôt distant

  • git remote add origin {adresse du GitHub}
  • git branch -M main
  • git push -u origin main

    • Commandes de gestion basique de branches

  • touch {nom du fichier} > creation de fichier
  • git add {nom du fichier} > stage les modifications (sauvegarde en dehors)
  • git commit -m “Ajout des fichiers html et css de base” > valide les modifications en locale sur la branche en cours
  • git push -u origin {nom de la branche sur le repository} > envoi des modifs sur le branche du serveur (repository)
  • git checkout > changer de branche
  • git merge {nom de la branche à fusionner} > fusionne la branche nommée avec celle 'checké'
